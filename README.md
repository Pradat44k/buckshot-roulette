# 🎯 Buckshot Roulette - Custom Keybinds

Script AutoHotkey v2 pour remapper les contrôles de **Buckshot Roulette** sur clavier.

## 🎮 Contrôles

| Touche | Action |
|--------|--------|
| `Q` | Tirer sur **soi-même** |
| `E` | Tirer sur le **dealer** |
| `1` | Item slot 1 |
| `2` | Item slot 2 |
| `3` | Item slot 3 |
| `4` | Item slot 4 |
| `5` | Item slot 5 |
| `6` | Item slot 6 |
| `7` | Item slot 7 |
| `8` | Item slot 8 |

## 🛠️ Utilitaires

| Touche | Action |
|--------|--------|
| `F1` | Mode Calibration ON/OFF |
| `F2` | Sauvegarder position souris |
| `F3` | Recharger le script |
| `F4` | Quitter le script |

## 📐 Comment calibrer

1. Lance le jeu **Buckshot Roulette**
2. Lance le script `BuckshotKeys.ahk`
3. Appuie sur `F1` pour activer le mode calibration
4. Place ta souris sur chaque item/position que tu veux mapper
5. Appuie sur `F2` pour noter les coordonnées
6. Modifie les positions dans `BuckshotKeys.ahk`
7. Appuie sur `F3` pour recharger le script

## 📋 Prérequis

- [AutoHotkey v2](https://www.autohotkey.com/)
- Buckshot Roulette
- Résolution 1920x1080 (modifiable dans le script)
